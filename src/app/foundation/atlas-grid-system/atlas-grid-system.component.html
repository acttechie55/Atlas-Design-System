Grid System

Our layout is based on the Bootstrap 12-column, responsive, mobile-first grid system.

Guidelines

Foundation of Our Grid System

The core of the Bootstrap's grid are rows and columns. Each row can have any amount columns totaling up to 12. Using classes, these columns can "break", allowing our content to respond to whatever viewport (mobile, tablet, or desktop) it appears.

Most of the time, we want our content to break at the tablet breakpoint. So you will probably use col-md-** more than any other grid class. However, feel free to use other grid classes when appropriate.

Note: You will rarely need to use col-**-12. Full-width, single column elements have no need for grid classes.

Breakpoints

Our layout grid includes five breakpoints—0, 576, 768, 992, and 1200—to configure across viewport widths.

Mobile-First

A mobile-first approach means that you start with the smallest breakpoint first and configure what changes as viewport width increases.
