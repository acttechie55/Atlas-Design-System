Spacing

Space is an integral tool in UX/UI design. It organizes content, influences readability, shapes visual hiearchy, and dictactes overall usability. Our goal is to apply space consistenly within and between components to render harmonious visual experiences.

CSS Utilities

Spacing can be modified by using one of the SASS variables or using a utility class.

Variables

ADS spacing variables follow the naming convention $atlas-spacing-base.

Each variable is a derivative of its base value (1 rem) and increases (or decreases) in quarter increments.

Sass Variable Name Value
{{variable.name}} {{variable.value}}

Utility Classes

Utility classes can be added directly to your html code. Built from a Sass mixin, classes are constructed ranging from 0rem to 4rem, increasing by quarter increments.

Syntax

To utilize our spacing utility classes, use the following format:

atlas-{{leftCurlyBraces}}property{{rightCurlyBraces}}-{{leftCurlyBraces}}direction{{rightCurlyBraces}}-{{leftCurlyBraces}}size{{rightCurlyBraces}}

Property stands for:

  • m - for classes that set margin
  • p - for classes that set padding

Direction stands for:

  • {{direction.abbreviation}} - for classes that set {{direction.className1}} {{direction.conjunction}} {{direction.className2}}
  • blank - for classes that set margin or padding on all 4 sides.

Sizes stands for:

  • {{w}} - {{direction.removeOrSet}} margin {{direction.conjunction}} padding {{direction.preposition}} {{direction.variable}}
  • auto - for classes that set the margin to auto.
